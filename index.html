📋 Règles du jeu

  1. Objectif : attraper le maximum de mouches avant la fin du chrono.
  2. Durée : chaque partie dure 60 secondes.
  3. Comment jouer : clique ou touche une mouche pour l'attraper.
  4. Points :
    • 🪰 Mouche classique → 10 pts
    • ⚡ Mouche rapide → 25 pts
    • ✨ Mouche dorée → 50 pts
  5. Combo : enchaîne les captures sans laisser une mouche s'échapper pendant 3 s → bonus x2 sur la prochaine capture.
  6. Difficulté : plus le temps passe, plus les mouches apparaissent et bougent vite.
  7. Pénalité : si une mouche disparaît toute seule, tu perds 5 points.
